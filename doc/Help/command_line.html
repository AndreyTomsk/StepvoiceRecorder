Command line

StepVoice Recorder supports the following command line parameters:

SvRec.exe /file="file_name.mp3"
Opens or creates a file (if not exist). You can use several predefined patterns for a directory and file name:

  • {Autoname} - specifies a program generated file name;
  • {Desktop} - desktop folder;
  • {MyDocuments} - documents folder;

SvRec.exe /record
Begins recording immediately after the program started.

SvRec.exe /close
Closes recorder. Can be useful if only one recorder instance allowed (see options).

Examples:
SvRec.exe /record /file="{Desktop}/Sample.mp3"
SvRec.exe /record /file="{MyDocuments}/{Autoname}.mp3"
