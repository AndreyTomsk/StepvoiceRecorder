How to Use?
Command Line Options

Stepvoice Recorder supports the following command line parameters:

/file="<file_name>"
Opens or creates a file (if not exist). You can also use several predefined patterns for a directory and file names:
- {Autoname} – auto file name, generated by the recorder.
- {Desktop} – desktop folder.
- {MyDocuments} – documents folder.

/record
Begins recording right after the program is started.

/close
Closes the program. This command is useful only when a single recorder instance allowed (see program preferences).


Example:

SvRec.exe /record /file="{Desktop}/{Autoname}.mp3"
