How to Use?
Recorder Preferences
To open the Stepvoice Recorder preferences dialog use the menu:
Options Preferences...

General Settings

The Startup options can be used to automatically create new or re-load a last opened file. If the Allow multiple instances is not set, then only a single instance of the recorder is started. This option is also used with the /close command line parameter.

Record Settings

This page shows current mp3 encoding parameters. Please see this topic for more details.

SAC Settings

This page shows current Sound Activation parameters. Please see this topic for more details.

Autoname Settings

In this tab you can set the preferred template for file naming. This template is used when the program creates a new file by Startup options (see General Settings) or command line parameters. Click on any template parameter (%y, %b, etc.) to insert it into the editor.
