How to Use?
Recording Source
Stepvoice Recorder uses a default sound device for audio recording. You can change it by opening the system audio properties dialog with the recorder menu:
Options Sound Devices...
Generally a sound card has several recording lines: microphone, line input, etc. To change a current line press the "Mixer Line Selection" button:

To record any sounds, currently playing on your computer, select the StereoMix line. This line is supported by most sound cards, but could have another name: What U Hear, Output, Mixer, Wave.

A lot of new computers running Windows Vista have a microphone line only. Try this solution to activate the StereoMix line: open the system audio properties dialog (see above), go to the Recording tab, right click in an empty area, and choose "Show Disabled Devices". Now, right click the stereo mix and choose "Set as standard".

The Stepvoice Recorder also provides the "What U Hear (Software)" virtual line for loopback recording.
