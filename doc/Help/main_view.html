Main View
The listing below specifies the Main Window parts:
  1. program state icon. It shows current state of the program: recording, playback, paused, stopped;
  2. visualization window. The following visualization types are supported: peak meter (linear), peak meter (logarithmic), wave meter. Right click in the window to open a context menu where you can select a visualization type;
  3. "create/open file" button;
  4. "record/pause" button. You can find detailed information about sound recording in the "How to..." section;
  5. "stop" button;
  6. "playback/pause" button;
  7. volume slider. Allows to adjust a current line's volume;
  8. mixer stream selection button. Opens the playback/recording stream selection menu;
  9. current playback/recording time;
  10. scheduler button. Enables/disables the program scheduler;
  11. voice activation button. Enables/disables voice activation;
  12. monitor input button. Shows current recording volume in the visualization window (see #2) to allow a user adjust the volume before recording;
  13. information window. Shows current MP3 settings and file informaion;
  14. seek slider. Shows a current playback/recording position.
