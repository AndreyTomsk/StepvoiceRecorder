Recording preferences
   You can run the recoder preferences dialog by menu "Options | Preferences..." . The dialog has several tabs:
  • General
  • Record
  • Scheduler
  • VAS
  • Autoname

General options

   By using the "Startup options" group you can open the last opened file or create a new one then the program starts. The new file is named automatically. It is created in the last opened file directory.

top

Recording options

   You can set mp3 encoding parameters here: mp3 bitrate, stereo mode and the recording frequency. The recording quality depends on the bitrate value. You can find the approximite sound quality and the one minute/hour file size in the text line under the bitrate slider.
You can also use the auto select fequency flag to let the program set the optimal recording frequency by itself.

top

Scheduler options

   The "Scheduled start" group allows to start recording automatically at the specified time. You can also select here any external file to run after the recording started (for example *.ram and *.pls files for internet radio recording).

top

Voice Activation options

    This function removes silent passages from a recording stream. If sounds are lower than Threshold Level during the Detection Time, the program pauses recording until a sound level is high enough to resume.

top

Autoname options

    The program generates default file names based on this template. Click on the template code to insert it in the template edit box.
