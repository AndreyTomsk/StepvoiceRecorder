Stepvoice Recorder
Overview

Welcome

Thank you for installing Stepvoice Recorder! Stepvoice Recorder is an advanced, yet user friendly sound recording solution for Microsoft Windows.


Main Features

  • Record from any source supported by a sound card: microphone, line in, etc.;
  • Record any playing sounds from the virtual "What U Hear" line in Vista;
  • Select an optimal sound source level before recording;
  • Automatically skip silent passages with Voice Activation;
  • Command line parameters.
