Stepvoice Recorder
What's New?

Version 1.9.1

  • Program executable and installer now digitally signed.
  • Faster display of the devices selection dialog.
  • Fixed saving of selected devices with same name (Phone devices).
  • Voice Activation feature renamed to "Sound Activation".

Version 1.9

  • Audio engine migrated to WASAPI (supported since Windows Vista).
  • Recording devices selection dialog.
  • Support recording from several lines at same time.
  • Result file is created automatically on recording start.
  • Scheduler removed, program can be controlled via command line parameters.
  • Lots of internal changes in code.

Version 1.8

  • Mixing virtual 'speakers' line with microphone (for Majicjack, Skype,...).
  • Added recording/playback device selection.
  • Current mixer line is restored on next start.
  • Adding VBR header to mp3 file.
  • Switched to 32-bit floating-point sample data.
  • Fixed program crash when starting in registered mode, Windows 7.
  • Many small bugs fixed.

Version 1.7

  • Added a virtual line to record playing sounds (loopback) in Windows Vista.
  • Added command line parameters.
  • Playback volume is changed via BASS functions.
  • Help file updated.
  • Fixed program crash after a reg. key entered from nag-screen.
  • Fixed recorder crash when entered incorrect symbols in Autoname dialog.
  • Fixed the "lame_end.dll not found" error.
  • Fixed a file open error, when it is already opened in another program.
  • Fixed the disabled "Play" button after a file is closed.
  • Fixed assertions when creating a new file by the "Record" button.

Version 1.6

  • Decoding library changed to Bass to fix playback in Vista.
  • Fixed incorrect play position update.
  • Fixed peak meter update speed depends on recording parameters.
  • Fixed wave meter freeze while playing.
  • Vista: Sound devices menu item.
  • Vista: Sound mixer didn't show the recording line icon.
