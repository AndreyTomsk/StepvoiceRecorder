FAQ (frequently asked questions)
  • How can i record streaming audio from the Internet (with Windows Media Player, Winamp, Real Player, etc.) ?
  • It records but the recorded file is silent. There is no movements on the sound level lines when recording.

   How can i record streaming audio from the Internet (with Windows Media Player, Winamp, Real Player, etc.) ?

   To record Internet streaming audio, you need to select "Stereo Mix" sound stream in the Windows mixer.

   After that start playing a radio stream in Winamp (Windows Media Player, Real Player, etc).

   Then start recording. Here you can find some advices, which help you to get better records.

   It records but the recorded file is silent. There is no movements on the sound level lines when recording.

   You need to select another recording stream in the program mixer. For example, if you want to record from a microphone, but the recording stream is "Line In", no sounds are recorded.
You can also use visualization window to detect recording.
