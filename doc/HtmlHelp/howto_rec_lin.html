How to record from an external audio device
  1. Connect a device to your sound card's Line In jack (usually blue);
  2. Create a new file and configure its mp3 parameters (current mp3 parameters are shown in the information window);
  3. Start playback in a device;
  4. Select the "Line In" stream in the program mixer and adjust its volume;
  5. Press the "Record" button to start recording.
