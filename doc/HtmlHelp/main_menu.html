Main Menu
   The program main menu consists of the following items:
  • File
  • Sound
  • Options
   Detailed information on each item is listed below. Nearly all menu items has shortcuts. They are shown in parentheses.

File
  • Create/Open... (Ctrl - O) - open a standard dialog box for creating/opening a new file;
  • Close - close currently opened file;
  • Clear (Ctrl - A) - clear file (set its size into zero);
  • Delete - delete file from a hard disk;
  • Find... (Ctrl - F) - open current file folder;
  • Exit - exit from the program.
top

Sound
  • Record (Enter) - start recording info a file. You can specify recording position by the seek slider. The detailed information about recording is listed here;
  • Stop (Shift - Enter) - stop recording/playback;
  • Play (Space) - play a file from current position;
  • Jump to begin (Home) - jump to the beginning of a file;
  • Back 5 seconds (Left) - jump to the 5 seconds back;
  • Fwd 5 seconds (Right) - jump to the 5 seconds forward;
  • Jump to end (End) - jump to the end of a file.
top

Options
  • Preferences... (Ctrl - P) - run preferences dialog box. You can change general and mp3 options here;
  • Always on Top (Ctrl - T) - makes the program window always on top;
  • System Mixer | Recording control... (Ctrl - R) - open the Windows mixer to select a recording stream;
  • System Mixer | Volume control... (Ctrl - V) - open the Windows mixer to change playback volume;
  • Sound Devices... - open sound devices setup window. You can select a default sound card here, which will be used for sound recording and playback.
