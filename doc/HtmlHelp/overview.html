Overview

   The StepVoice Recorder is full-featured and award winning software for sound recording. It allows you to record from any sound source, like microphone or internet streaming audio. The record is compressed in real time so you can record sounds without running out of memory until your hard disk is full.

Features:
  • Easy of use and friendly interface;
  • System tray support;
  • Ability to record from any sound sources, such as microphone, line in, CD player, internet streaming audio, etc.;
  • Supporting audio formats: MP3;
  • "On-the-fly" mp3 encoding (without temporary files);
  • Playback/Recording graphical visualization with logarithmic peak meter;
  • Recording into any part of a file;
  • File operations: delete, clean, etc.;
  • NEW: Built-in mixer;
  • NEW: Schedule recordings;
  • NEW: Voice activation system;
  • NEW: Sound monitoring;
  • NEW: Advanced file autoname system;
