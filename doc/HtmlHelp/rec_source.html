How to record from different sound sources
   You should use the Windows Mixer for selecting a sound stream, from which StepVoice Recorder will record sounds. Here are some sources you can record from:
  • Microphone
  • External sound devices, such as videotape, walkman, and so on
  • Internet audio streams, movie sounds, etc.

   To run the mixer use menu "Options | Windows Mixer | Recording control..." or press button in the main program window.

Here is a typical mixer window:

  • To record from microphone select the "Microphone" stream.

  • To record from external sound devices select the "Line In" stream.

  • For Internet audio recording select the "Stereo Mix" stream. Producers name this stream differently, so here are some other possible names: "What U hear", "Output", "Mixer", "Wave", "Sum".

Note.
   You need a fullduplex sound card for recording internet audio (generally any sounds from computer) - it should play and record sounds simultaneously. Contact your card producer if a card supports a fullduplex mode, and if needed, download latest drivers for it.
