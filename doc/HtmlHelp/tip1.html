How to get optimal records

    To get optimal sound quality it is nesessary to avoid of too low and too high recording level (in the second case distortions appears). To adjust the level run the Input Monitor (click "M" button in the main window) first, then select a sound stream you need by the Mixer Stream Selection button and adjust its level until maximum peaks are about 70..80% (-3..-2dB):

    To keep your hard disk space you can select a suitable mp3 parameters before recording. For example - you want to record an online radio stream from Winamp:

    In the Winamp window you can see the radio stream parameters: bitrate, frequency and stereo mode. You need to select the same parameters in the StepVoice Recorder: bitrate = 32 kbps, frequency = 22.050 KHz, mono. In this case one hour recording requires 14.1 Mb of hard disk space.

    If you set a higher bitrate, then a record will be the same quality but take more hard disk space.

Note.
    If you set stereo mode instead of mono, then sound quality will be lower, because every channel (left and right) has only 32/2 = 16 kbps. Thus, to get higher quality on low bitrates it is better to record in mono mode.
