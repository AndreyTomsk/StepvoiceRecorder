
12.07.2008 Мастер настройки записи.

Предназначен для решения проблемы выбора линии для записи. Мастер преставляет собой диалог, запускаемый из меню "Options". Он должен поддерживать возможность автоматического и ручного выбора источника записи.

Должен содержать элементы:
1. текстовое пояснение для пользователя;
2. график визуализации;
3. выбор звуковой карты и ее линии;
4. кнопка запуска автоматического сканирования;
5.

12.07.08 Диалог автоматического сканирования.

Представляет собой упрощенный вариант мастера настройки записи. При запуске включает Мониторинг и начинает сканирование всех линий


13.07.08 Управление программой с командной строки.

Параметры:
/close			- закрыть приложение;
/file:[<path to file>]	- создать файл для записи;
/record			- начать запись

Пример:
SvRec.exe /?
SvRec.exe /record [/file:{Desktop}/{Autoname}.mp3]
ScRec.exe /close

/file:"{Desktop}"
/file:"{MyDocuments}"
/file:"{LastUsed}"
/file:"c:\some dir\"

Если разрешен запуск только одной копии программы, то параметры передаются ей.
Иначе обрабатываем их в нашей программе. В этом случае также запускаем окно в свернутом виде.

Для триальной версии сделать нагскрин в правой нижней части окна. Программа будет записываться с текущими параметрами.
